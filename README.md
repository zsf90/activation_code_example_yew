# 激活码练习项目

项目功能：

输入一个 4 位整数，更具简单算法算出激活码。

该项目很简单，只要一个输入框和一个按键即可，在 输入框中输入4位整数，点击生成按钮生成激活码。

## 项目 src 目录结构

```shell
D:.
└─components
    ├─atoms
    ├─molecules
    ├─organisms
    └─pages
```

* atoms 代表不可分的最小元素
* molecules 翻译过来是“分子”的意思，比 atoms 大一些的元素
* organisms 翻译过来是“有机物”的意思，用来存放导航，等有其他更小的元素组成的组件
